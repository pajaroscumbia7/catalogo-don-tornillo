# Don Tornillo — Catálogo

Catálogo web de **Don Tornillo** (tornillería, corte y grabado láser en Corregidora, Querétaro).
Es un solo archivo `index.html`, autocontenido: las fotos van embebidas dentro, así que no depende de nada externo y se publica gratis en segundos.

## Publicarlo gratis con GitHub Pages

1. Crea un repositorio nuevo en GitHub (botón **New**). Nómbralo, por ejemplo, `catalogo-don-tornillo` y márcalo **Public**.
2. En el repo vacío: **Add file → Upload files**. Arrastra `index.html` (y este `README.md`) y da **Commit changes**.
3. Ve a **Settings → Pages**. En *Build and deployment* → *Source*: **Deploy from a branch**; *Branch*: **main** y carpeta **/ (root)**. Guarda.
4. Espera ~1 minuto. GitHub te dará un enlace como
   `https://TU-USUARIO.github.io/catalogo-don-tornillo/`
   Ese es tu catálogo en vivo, listo para compartir por WhatsApp y Facebook.

## Actualizarlo

- Para cambiar productos, precios o fotos: reemplaza `index.html` con **Add file → Upload files** (sube el archivo nuevo con el mismo nombre y haz commit).
- Si quieres, pídeme que te regenere el `index.html` con los cambios y solo lo vuelves a subir.
